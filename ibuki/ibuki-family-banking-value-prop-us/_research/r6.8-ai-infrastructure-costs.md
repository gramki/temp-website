# Research: US AI/ML Infrastructure Costs

**Research ID:** R6.8  
**Date:** 2024-12-19  
**Purpose:** value-to-bank.md - Business Case

## Research Question

What are LLM API costs, compute costs, and model training costs for US banking AI agents?

## Key Findings

US AI infrastructure costs: LLM APIs ($0.002-$0.01 per 1K tokens), compute ($0.10-$0.50 per hour), model training ($10K-$100K one-time). Annual costs scale with usage.

## Data Points

| Component | Cost | Unit | Source |
|-----------|------|------|--------|
| LLM API (GPT-4) | $0.01-$0.03 | Per 1K tokens | OpenAI |
| LLM API (GPT-3.5) | $0.002-$0.0015 | Per 1K tokens | OpenAI |
| Compute (GPU) | $0.50-$2.00 | Per hour | Cloud providers |
| Model Training | $10K-$100K | One-time | Industry estimates |
| Annual Infrastructure | $2M-$8M | Per 100K customers | Calculated |

## Detailed Findings

### LLM API Costs

**Usage Patterns:**
- Average conversation: 500-2,000 tokens
- Cost per conversation: $0.01-$0.06
- Monthly conversations: 10-20 per customer
- Monthly cost per customer: $0.10-$1.20

**Annual Cost:**
- Per customer: $1.20-$14.40
- Per 100K customers: $120K-$1.44M

### Compute Costs

**Infrastructure:**
- GPU instances for inference
- Data storage
- Network costs
- Annual: $500K-$2M for 100K customers

### Model Training

**One-Time Costs:**
- Fine-tuning: $10K-$50K
- Custom models: $50K-$100K
- Ongoing updates: $10K-$20K annually

## Relevance to Proposal

AI infrastructure costs are significant but scale with usage. Cost per customer decreases as adoption grows. This is a key component of investment and ongoing costs.
