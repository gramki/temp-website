# Research: US Banking Integration Costs

**Research ID:** R6.9  
**Date:** 2024-12-19  
**Purpose:** value-to-bank.md - Business Case

## Research Question

What are core banking integration and payment rail integration costs for US banks?

## Key Findings

US banking integration costs: core banking ($500K-$2M one-time), payment rails ($200K-$500K one-time), APIs ($100K-$300K one-time). Ongoing maintenance: $200K-$500K annually.

## Data Points

| Integration | One-Time Cost | Annual Maintenance | Source |
|-------------|---------------|-------------------|--------|
| Core Banking | $500K-$2M | $200K-$400K | Industry estimates |
| Payment Rails | $200K-$500K | $100K-$200K | Industry estimates |
| APIs | $100K-$300K | $50K-$100K | Industry estimates |
| **Total** | **$800K-$2.8M** | **$350K-$700K** | Calculated |

## Detailed Findings

### Core Banking Integration

**Components:**
- API development
- Data mapping
- Security implementation
- Testing
- Cost: $500K-$2M

**Variation:**
- Legacy systems: Higher cost
- Modern APIs: Lower cost
- Custom requirements: Higher cost

### Payment Rails

**ACH Integration:**
- Network connections
- API development
- Testing
- Cost: $100K-$200K

**Wire Transfer:**
- Network connections
- Security requirements
- Cost: $50K-$150K

**Zelle:**
- Bank network integration
- Cost: $50K-$150K
