Org 8.0 — Documentation Index

This folder contains the main organizational and product documentation for Org 8.0. Use this page as an index to navigate to each area.

Engagement Operations Manual

The Engagement Operating Model Guide is the practitioner's guide to how Engagements are explored, staffed, executed, and governed. It covers roles, lifecycle phases, squad model, architecture practices, operating models, and governance.

→

Product Line Engineering

Product Line Engineering (PLE) describes Zeta's mental model and operational framework for software delivery: core asset development (Product Line Squads), Engagement Engineering (Customer Product delivery), governance (Platform Architecture Council, inner source, variability), and adoption.

→

Product Information Model

The Unified Product Information Model (UPIM) is a framework for describing, managing, and evolving a software product. It provides a shared language across Product Management, Engineering, Operations, and Business, structured as a Definition Model, Work Model, and Operating Model.

→

Domain Products with DDD and Spring Boot

A developer and Principal Engineer guide for implementing Domain-Driven Design (DDD) with Spring Boot: mapping product requirements to DDD, one application per domain, no ORM, and keeping the domain explicit and testable.
