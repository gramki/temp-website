Initiative Scoping Task

Model: Work Model
Track: Track 1: The Discovery Track (Learning)
Owner: Product Management

Definition

Work to define an Initiative's scope, success criteria, target outcomes, and timeline. Includes identifying which Signals (Problems, Needs, Opportunities) should be associated with the Initiative.

Purpose

Makes the Initiative definition work explicit. An Initiative (Dimension 1) requires deliberate scoping — determining boundaries, success metrics, target customers, and expected outcomes — before it can guide prioritization of Signals for discovery.
