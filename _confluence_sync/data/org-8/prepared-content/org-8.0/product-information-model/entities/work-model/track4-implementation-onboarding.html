Implementation / Onboarding

Note: Implementation / Onboarding is a subtype of Win Activity. See track4-win-activity.md for the parent entity and other subtypes.

Model: Work Model
Track: Track 4: The Win Track (Value Realization)
Category: Activity
Owner: Customer Success, Sales, Implementation Consultants

Definition

Steps to integrate a specific client — the work required to get a new customer from contract signing to active usage.

Purpose

Captures the per-customer activation work. Implementation/Onboarding is where the product's value is actualized for a specific buyer.
