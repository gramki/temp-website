# Accountability Guide: Full Narrative Rewrite

## Target File

[org-8.0/people-and-culture/accountability.md](org-8.0/people-and-culture/accountability.md)

## Approach

- Rewrite in place, section by section
- Each section is an independent task with full context from the original doc
- Voice: external coach, practical playbook, prescriptive
- Audience: unified -- readable by all employees, actionable for managers
- References: public coaching/leadership material (Brown, Lencioni, Raymond, Scott, Edmondson, Connors et al.)
- Executive Summary written last, summarizing what was actually produced

## Module Map (12 sections, written in order)

### Task 1: Module 1 -- What Accountability Actually Is (And Isn't)

- Reframe accountability away from blame
- Opening scenario (two teams)
- Core definition: ownership, trust, role
- References: Brene Brown *Dare to Lead*, Lencioni *Five Dysfunctions*

### Task 2: Module 2 -- The Two Engines (Outcomes vs. Commitments)

- Side-by-side scenario illustrating the distinction
- Comparison table
- Why conflating them is dangerous
- Reference: Roger Martin *The Responsibility Virus*

### Task 3: Module 3 -- Making Accountability Real (Setting Expectations)

- How to articulate accountability for teams and individuals
- Starter phrases and example statements
- Reference: *The Oz Principle* (Connors, Smith, Hickman)

### Task 4: Module 4 -- The Manager's Enforcement Playbook

- 4A: Outcome Misses -- the Learning Loop (Diagnose, Correct, Monitor)
- 4B: Commitment Misses -- the Reliability Loop (Transparency, Containment, Consequences)
- Escalation matrices for teams and individuals (tables retained, narrated)

### Task 5: Module 5 -- Having the Conversation

- Jonathan Raymond's Accountability Dial (*Good Authority*)
- 1-2 scripted conversation examples at different escalation levels
- Anti-patterns: what not to say, with alternatives
- Reference: Kim Scott *Radical Candor*

### Task 6: Module 6 -- Team vs. Individual (Knowing Where the Problem Lives)

- Diagnostic guidance: is it a system failure or an individual failure?
- Why the distinction matters
- Diagnostic checklist

### Task 7: Module 7 -- Upward Accountability

- Prescriptive: what we expect at this organization
- How managers hold their own leadership accountable
- Reference: Amy Edmondson *The Fearless Organization*

### Task 8: Module 8 -- Accountability and Career Growth

- Link between accountability, trust, scope expansion, and promotion
- "Earning bigger problems" framing

### Task 9: Module 9 -- Non-Negotiable Principles for Managers

- Five principles from original doc, narrated with 1-2 examples each
- Prescriptive tone

### Task 10: Module 10 -- Failure Modes (Signs Your Team Has a Problem)

- Expanded from original Section 7
- Each mode: symptom, root cause, corrective action

### Task 11: Module 11 -- Quick Reference (Practical Operating Rules)

- One-page cheat sheet
- Refined from original Section 8

### Task 12: Module 0 -- Executive Summary

- Written LAST after all modules are complete
- Summarizes actual content produced
- ~300 words, standalone TL;DR

### Task 13: Final Assembly and Review

- Ensure narrative flow between modules
- Check cross-references and consistency
- Verify all cited references are accurate

## Key References to Cite

- Brene Brown, *Dare to Lead* (2018)
- Patrick Lencioni, *The Five Dysfunctions of a Team* (2002)
- Roger Martin, *The Responsibility Virus* (2002)
- Connors, Smith, Hickman, *The Oz Principle* (1994/2004)
- Jonathan Raymond, *Good Authority* (2016) -- the Accountability Dial
- Kim Scott, *Radical Candor* (2017)
- Amy Edmondson, *The Fearless Organization* (2018)
