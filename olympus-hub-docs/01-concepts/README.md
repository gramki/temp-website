# Core Concepts

This section contains the foundational concepts that define how Hub models and operates on information-centric work.

---

## What's Here

**Core Concepts** define *what* Hub is and *how* it thinks about work. This is distinct from **System Design** (in `02-system-design/`), which covers *how* Hub is built.

| Document Type | What It Covers |
|---------------|----------------|
| **Ontology** | The four-layer conceptual framework (Perception, Normative, Execution, Automation) |
| **Work Patterns** | The seven fundamental patterns of information-centric work |
| **Terminology** | Key terms that bridge Hub concepts to industry language |
| **Integration** | How Hub connects to external systems and agents |
