## Overview

The **Customer Success Executive** onboards tenants, supports their success, and ensures productive use of the Hub platform. They operate at the Hub System scope (Publisher Domain).

| Attribute | Value |
|-----------|-------|
| **Category** | Hub Persona — Hub System (Publisher) |
| **Scope** | Hub System |
| **Domain** | Publisher Identity Domain (Zeta) |
| **Primary Console** | Customer Success Portal |

---

## Objectives

| Objective | Description |
|-----------|-------------|
| **Onboard Tenants** | Create subscriptions, guide initial setup |
| **Support Success** | Ensure productive platform usage |
| **Review Usage** | Monitor tenant adoption and value |
| **Provide Assistance** | Help with operational questions |

---

## Key Activities

### Onboarding

1. **Subscription Creation**
   - Create new tenant subscriptions
   - Configure initial resources
   - Provision administrator accounts

2. **Setup Guidance**
   - Guide through configuration process
   - Connect with documentation
   - Coordinate with SRE for technical needs

### Ongoing Support

1. **Usage Review**
   - Monitor adoption metrics
   - Identify underutilization
   - Recommend optimizations

2. **Relationship Management**
   - Regular check-ins
   - Feature education
   - Escalation coordination
