## Overview

The **Administrator** manages the tenant subscription — resources, users, machines, and overall configuration. They are the operational owner of the tenant's Hub presence.

| Attribute | Value |
|-----------|-------|
| **Scope** | Tenant Subscription |
| **Domain** | Tenant Identity Domain |
| **Primary Console** | Subscription Portal |

---

## Objectives

| Objective | Description |
|-----------|-------------|
| **Manage Subscription** | Configure resources, budgets, branding |
| **Manage Users** | Provision and manage user access |
| **Configure Resources** | Set up data stores, machines, I/O gateways |
| **Coordinate Support** | Work with Hub SRE and Customer Success |

---

## Key Activities

### Subscription Setup

1. **Resource Provisioning**
   - Allocate data stores (Ganymede, Callisto, Europa)
   - Configure memory and knowledge stores
   - Set up I/O gateways

2. **User Management**
   - Define identity domains
   - Provision users by role
   - Configure access policies

3. **Machine Registration**
   - Register external machines
   - Configure tool access
   - Set up credentials and policies

### Ongoing Operations

1. **Budget Monitoring**
   - Track resource usage
   - Manage cost allocations
   - Review billing

2. **Support Coordination**
   - Request technical assistance
   - Coordinate with Customer Success
   - Handle escalations
