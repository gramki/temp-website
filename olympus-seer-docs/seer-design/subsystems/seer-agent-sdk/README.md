# Seer Agent SDK

> **Status**: 🟡 Draft — Capability outline  
> **Last Updated**: 2026-01-11

## Overview

Seer Agent SDK provides the SDK for Raw Agents to interact with Seer and Hub services. The SDK provides APIs for employment spec access, prompt access, context compilation, metrics reporting, tracing, tool discovery/calling, memory services, knowledge services, and events.

---

## Capabilities

Based on `olympus-hub-docs/scratchpad/seer-subsystems.md`:

- Employment Spec Access APIs
- Prompt Access APIs (A/B Testing Aware, Authority Enforcement aware)
- Context Compiler APIs
- Metrics Reporting APIs
- Trace APIs
- Hub Tool Discovery APIs
- Hub Tool Calling APIs
- Hub Stores APIs
- Hub Knowledge Services APIs
- Hub Enterprise Memory Services APIs
- Hub Agent Memory Services APIs
- Hub Events APIs
- Convenience APIs for Building LangGraph Agent, StrandsAgent, OpenAPI Agent in a Seer Environment
