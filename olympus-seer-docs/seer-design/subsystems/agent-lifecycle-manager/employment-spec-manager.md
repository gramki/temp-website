# Employment Spec Manager

> **Status**: 🟡 Draft — Capability outline  
> **Last Updated**: 2026-01-11

## Overview

Employment Spec Manager manages Employment Specifications for Employed Agents, including authority enforcement controls, resource quotas, fair usage budgets, and delegation chain configuration.

---

## Capabilities

- Authority Enforcement Controls
- Resource Quota management
- Fair Usage Budget (Per Subject, Per Signal, etc.)
- Delegation Chain configuration
