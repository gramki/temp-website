# Context Compiler

> **Status**: 🟡 Draft — Capability outline  
> **Last Updated**: 2026-01-11

## Overview

Context Compiler provides the service implementation for context assembly, compiling context from Enterprise Knowledge, Enterprise Memory, and Agent Memory sources.

---

## Capabilities

Based on `olympus-hub-docs/scratchpad/seer-subsystems.md`:

- Context compilation from three sources (Enterprise Knowledge, Enterprise Memory, Agent Memory)
- Context ranking and relevance scoring
- Token budgeting and truncation
- Context provenance tracking
- Reproducible context assembly
