# Cipher IAM Extensions for Agents

> **Status**: 🟡 Draft — Capability outline  
> **Last Updated**: 2026-01-11

## Overview

Cipher IAM Extensions for Agents extends the Hub Cipher IAM system to support agent-specific identity and authority management. This subsystem provides the IAM foundation for agent identity, authority delegation, and policy enforcement.

---

## Capabilities

Based on `olympus-hub-docs/scratchpad/seer-subsystems.md`:

- Raw, Trained, and Employed Agent Profile Tags
- Authority Delegation
- Human Accountability
- Policies Per Policy Enforcement Point (Resource Provider) registered with Cipher IAM
