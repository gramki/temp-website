# Iteration Planning Task

**Model:** Work Model
**Track:** Track 2: The Build Track (Construction)
**Owner:** Tech Lead, Developers

## Definition

Work to assign Epics/Stories to a time-boxed iteration (sprint), balance team capacity, and identify dependencies and risks for the iteration.

## Purpose

The tactical planning unit within the Build Track. Iteration Planning Tasks ensure that the right work is pulled into each sprint, capacity is balanced, and teams are aligned on what will be delivered.
