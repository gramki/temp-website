# Technical Task

**Model:** Work Model
**Track:** Track 2: The Build Track (Construction)
**Owner:** Developers

## Definition

A granular engineering step — a specific coding, testing, or infrastructure action required to implement a User Story or Epic.

## Purpose

The most granular work entity in the Build Track. Technical Tasks represent the actual developer-level work: writing code, writing tests, configuring infrastructure, etc.
