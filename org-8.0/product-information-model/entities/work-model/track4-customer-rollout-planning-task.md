# Customer Rollout Planning Task

**Model:** Work Model
**Track:** Track 4: The Win Track (Value Realization)
**Owner:** Customer Success, Product Marketing

## Definition

Work to plan phased customer rollout — target segments, sequencing, migration support, and success criteria for each phase. Determines which customers receive the new capabilities first and the gating criteria for expanding to subsequent segments.

## Purpose

Not all customers should receive new capabilities simultaneously. Phased rollout reduces risk, allows for early feedback from trusted customers, and ensures support teams can handle adoption at each phase. This entity makes that planning work explicit.
