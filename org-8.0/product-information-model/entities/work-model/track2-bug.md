# Bug

**Model:** Work Model
**Track:** Track 2: The Build Track (Construction)
**Owner:** QA, Developers

## Definition

Unplanned defect resolution — a report of incorrect, unexpected, or broken behavior that requires a fix.

## Purpose

Captures unplanned work within the Build Track. Bugs disrupt the planned delivery of Epics and User Stories and must be triaged, prioritized, and resolved.
