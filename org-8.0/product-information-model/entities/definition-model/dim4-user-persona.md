# User Persona

**Model:** Definition Model
**Dimension:** Dimension 4: The User-Centric Dimension (Experience)
**Owner:** UX Designers, Product Management

## Definition

The human interacting with the system. A User Persona represents a distinct role or archetype that uses the product's UI-driven modules.

## Purpose

Anchors the Experience dimension by identifying *who* interacts with the product. User Journeys and Touchpoints are designed for specific personas.
