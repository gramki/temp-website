# Data Entity

**Model:** Definition Model
**Dimension:** Dimension 9: The Data & Information Dimension (State & Flow)
**Owner:** Enterprise Architects, Developers

## Definition

A persistent object stored by a specific Module — a database table, document, or record type with defined cardinality relationships to other entities.

## Purpose

Defines the information the system persists. Data Entities contain Attributes/Fields and progress through States.
