# User Journey

**Model:** Definition Model
**Dimension:** Dimension 4: The User-Centric Dimension (Experience)
**Owner:** UX Designers, Product Management

## Definition

The end-to-end path a User Persona takes to achieve a specific goal within the product.

## Purpose

Maps the sequential experience of a user across the product, from intent to completion. Journeys contain Touchpoints as their constituent interaction steps.
