# Pricing Tier / Package

**Model:** Definition Model
**Dimension:** Dimension 2: Business Value Dimension (Vendor Economics)
**Owner:** Product Marketing, Executives

## Definition

A bundled offering sold to a specific market segment. A Package groups Features into a marketable unit with associated pricing.

## Purpose

Connects the business model to the product's functional reality by defining which Features (Dimension 8) are included in each commercial offering.
