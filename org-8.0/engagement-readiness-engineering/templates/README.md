# Templates

[← Back to ERE Guide](../README.md)

> [!NOTE]
> Templates will be added in a future iteration.

This folder will contain document templates for:

- **Engagement artifacts** — charter, retrospectives, case studies
- **PI planning** — objectives, backlog, risks, retrospective
- **Requirements** — ADRs, change requests, gap analysis
- **Knowledge capture** — pattern documentation, lessons learned

Templates will be:
- Pre-populated with standard structure and guidance
- Validated by ERE tooling for completeness
- Available through the Bootstrap Kit

---

## Planned Templates

| Template | Purpose | Used By |
|----------|---------|---------|
| `engagement-charter.md` | Engagement kickoff and scope | EPM, EO |
| `adr-template.md` | Architecture Decision Records | EA |
| `change-request.md` | Scope change documentation | EPM, EPO |
| `pi-objectives.md` | PI planning objectives | EPM, EPO |
| `pi-retrospective.md` | PI retrospective | EPM |
| `case-study.md` | Engagement case study | EPM, Knowledge Engineer |
| `pattern.md` | Reusable pattern documentation | EA, Knowledge Engineer |
| `lessons-learned.md` | Phase transition learnings | EPM |
