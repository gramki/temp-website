1. Hub is a Business Domain of an Enterprise (typically, a workbench in the Olympus Hub Story)
2. Stream is a set of Scenarios that collectively represent a Value Stream that an Enterprise delivers to its Customers or Users
3. Loops are virtuous loops that can be created to improve value and reduce cost in a Business Domain
