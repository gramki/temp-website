# Work Catalog User Guide

This guide covers how to browse, author, test, and publish Work Catalog content.

## Audience

| Role | Primary Interest |
|------|------------------|
| Builders (developers, designers, QA) | Browsing available scenarios, understanding work flow |
| Workbench Managers | Customizing scenarios for their project |
| Workshop Admins | Setting team-wide defaults and standards |
| Foundry Admins | Establishing organization-wide patterns |

## Quick Start Paths

**"I want to understand how work flows through the system"**
→ Start with [how-work-flows.md](how-work-flows.md)

**"I want to see what scenarios are available for my workspace"**
→ See [browsing-catalogs.md](browsing-catalogs.md)

**"I want to create a custom scenario for my project"**
→ Read [authoring-scenarios.md](authoring-scenarios.md), then [testing-scenarios.md](testing-scenarios.md), then [publishing-workflow.md](publishing-workflow.md)

**"Something isn't working as expected"**
→ Check [troubleshooting.md](troubleshooting.md)
