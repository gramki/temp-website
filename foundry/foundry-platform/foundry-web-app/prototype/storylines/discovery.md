# Discovery Storyline (Prototype v0.1)

## Goal

Demonstrate how a Product Manager initiates and advances a Discovery Case to the
proceed-to-build decision.

## Walkthrough steps

1. Enter Workbench and open Discovery-oriented console.
2. Create Discovery Case with core metadata.
3. Review stage progression and active Work Orders.
4. Inspect blocker or governance indicators if present.
5. Complete decision gate with `proceed-to-build`.
6. Confirm Product Intent handoff visibility.

## Minimum screens

- Discovery list view
- Discovery detail view
- Decision gate view
- Handoff confirmation view

## Demo scenarios

- Happy path
- Blocked by missing evidence
- Reopened case after decision rejection
