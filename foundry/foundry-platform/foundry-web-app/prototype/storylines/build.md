# Build Storyline (Prototype v0.1)

## Goal

Demonstrate how teams progress a Product Intent through Build stages with Work
Order visibility and governance context.

## Walkthrough steps

1. Open Build-focused console for a Product Intent.
2. Review current stage and pending Work Orders.
3. Inspect Work Order detail and task status.
4. Observe quality or governance indicators.
5. Advance stage when criteria are satisfied.
6. Review updated traceability chain.

## Minimum screens

- Product Intent list
- Product Intent detail
- Work Order board/list
- Governance status view
- Traceability view

## Demo scenarios

- Happy path with sequential Work Orders
- Parallel Work Orders with one delayed branch
- Hard block at governance checkpoint
