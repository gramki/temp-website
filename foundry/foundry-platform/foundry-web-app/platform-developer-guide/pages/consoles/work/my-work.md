# My Work Console

The My Work console provides a personal view of the current user's assigned work across all workspaces.

## Type

Landing (Personal)

## Audience

Individual Workbench users

## Sections

### My Day

Today's focus:
- Work Orders due today
- Scheduled meetings/rituals
- Blockers requiring attention
- Active Workspace Sessions (with View session drill-down)

### My Week

Weekly view:
- Work Orders due this week
- Upcoming rituals
- Planned capacity
- Progress against commitments

### My Month

Monthly perspective:
- Work Order completions
- Contribution summary
- Upcoming deadlines
- Capacity planning

### Quick Actions

- Start Workspace Session
- View session details
- Pick up next Work Order
- Update Work Order status
- Log time

## Actions

| Action | Description |
|--------|-------------|
| Start session | Launch new Workspace Session |
| View session | Open canonical [Workspace Session Details](../workspaces/workspace-session-details.md) page |
| View Work Order | Open Work Order detail |
| Update status | Change Work Order status |
| Mark complete | Complete a Work Order |
| Log time | Record time against Work Order |

## URL

`/workbenches/{workbenchId}/consoles/my-work`
