# Foundry Web App user guide

This guide covers persona-specific journeys and tasks in the Foundry Web App — the primary web interface for Foundry members and Foundry Admins.

## Audience

| Role | Primary interest |
|------|------------------|
| Developers | Development Workspace tasks and Work Orders |
| Product Managers | Product Specification Workspace |
| Quality Teams | QA Workspace |
| UX Teams | UX Design Workspace |
| Release Teams | Release Workspace and CI/CD status |
| Governance Teams | Governance Workspace |
| Foundry Admins | Foundry Management admin console |
