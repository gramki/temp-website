# Workforce Overview Console

The Workforce Overview console provides a landing page for Workforce, showing combined team and agent summary with capacity information.

## Type

Landing

## Audience

Engineering Managers, Team Leads, Governance

## Sections

### Team Summary

- Total team members
- Active members (with sessions)
- Members by role
- Capacity utilization

### Agent Summary

- Total active agents
- Agent utilization
- Tasks by agent type
- Cost summary

### Capacity Overview

- Total capacity (human + agent)
- Current allocation
- Bottlenecks
- Capacity forecast

### Recognition Highlights

Recent Kudos and recognition:
- Top contributors this period
- Recent Kudos given
- Recognition trends

## Actions

| Action | Description |
|--------|-------------|
| View Team | Navigate to Team Console |
| View Agents | Navigate to Agent Console |
| Give Kudos | Open Kudos form |
| View capacity details | Drill into capacity analytics |

## URL

`/workbenches/{workbenchId}/consoles/workforce-overview`
