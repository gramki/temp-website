# Reports Console

**URL pattern:** `/workbenches/{workbenchId}/consoles/reports`

**Group:** Governance

**Purpose:** Generated reports, audit exports, and scheduled reporting.

---

## Page Sections

### 1. Report Library

| Report Type | Description |
|-------------|-------------|
| **Progress Report** | Work completion summary |
| **Quality Report** | Test results, coverage |
| **Release Report** | Deployment history |
| **Risk Report** | Active risks, mitigations |
| **Team Report** | Contribution analytics |
| **Agent Report** | Agent performance |
| **Audit Report** | Compliance evidence |

### 2. Recent Reports

| Column | Description |
|--------|-------------|
| Report | Name/type |
| Generated | When created |
| Period | Date range covered |
| Generated by | User or scheduled |
| Format | PDF, CSV, etc. |
| Actions | View, download, share |

### 3. Scheduled Reports

| Element | Description |
|---------|-------------|
| **Report name** | Identifier |
| **Type** | Which report |
| **Schedule** | Daily, weekly, monthly |
| **Recipients** | Who receives it |
| **Last run** | Most recent generation |
| **Next run** | Upcoming scheduled time |

### 4. Report Builder (Custom Reports)

| Element | Description |
|---------|-------------|
| **Data sources** | Select what to include |
| **Filters** | PI, Workspace, date range |
| **Columns** | Choose fields |
| **Grouping** | How to aggregate |
| **Format** | Output format |
| **Preview** | See before generating |

### 5. Audit Exports

| Export Type | Description |
|-------------|-------------|
| **Work Order history** | Complete WO audit trail |
| **Gate decisions** | Governance approvals/rejections |
| **Change history** | Repository changes |
| **Access logs** | Who accessed what |

---

## Actions

| Action | Who | Description |
|--------|-----|-------------|
| Generate report | All | Create a report now |
| Schedule report | Manager | Set up recurring reports |
| Download | All | Get report file |
| Share | All | Send to others |
| Delete | Manager | Remove old reports |
| Create custom | Manager | Build custom report |

---

## Report Formats

- **PDF** — Formatted for printing/sharing
- **CSV** — Data export for analysis
- **Excel** — Spreadsheet with charts
- **JSON** — Structured data export

- By component
