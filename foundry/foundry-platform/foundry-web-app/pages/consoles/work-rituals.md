# Work Rituals Console

The Work Rituals console manages team operating cadences: standups, sprint ceremonies for Build track, and kanban boards for Discovery, Run, Win, and Evolve tracks.

## Type

Workflow

## Audience

All Workbench team members

## Sections

### Standup View

Daily standup board showing:
- Yesterday's completions
- Today's focus
- Blockers
- Per-person status

### Sprint Board (Build Track)

Sprint-based view for Development work:
- Current sprint backlog
- Sprint burndown
- Velocity trend
- Sprint goal status

### Kanban Boards (Discovery, Run, Win, Evolve)

Kanban-style boards for non-sprint tracks:
- Columns by workflow stage
- WIP limits
- Cycle time indicators
- Flow metrics

### Ritual Calendar

Upcoming ceremonies:
- Sprint Planning
- Sprint Review
- Retrospective
- Daily Standups

## Actions

| Action | Description |
|--------|-------------|
| Start standup | Begin daily standup ritual |
| Move card | Drag Work Order between columns |
| Set sprint goal | Define current sprint objective |
| Schedule ritual | Add ritual to calendar |
| View past rituals | Review previous standup/sprint notes |

## URL

`/workbenches/{workbenchId}/consoles/work-rituals`
