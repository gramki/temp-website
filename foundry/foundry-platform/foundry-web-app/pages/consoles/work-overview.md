# Work Overview Console

The Work Overview console is the landing page for Work, providing attention queues, blocked items, due dates, and quick navigation.

## Type

Landing

## Audience

All Workbench users

## Sections

### Attention Queue

Items requiring immediate attention:
- Blocked Work Orders
- Items past due
- Items due soon
- Pending approvals

### Navigation Shortcuts

Quick links to:
- Orchestration
- Progress
- Rituals
- My Work

### Summary Metrics

- Active orchestration items by type
- Work Orders in progress
- Items blocked across workspaces

## Actions

| Action | Description |
|--------|-------------|
| Navigate to item | Open blocked/due item in context |
| View Orchestration | Go to Orchestration console |
| View Progress | Go to Progress console |
| View Rituals | Go to Rituals console |

## URL

`/workbenches/{workbenchId}/consoles/work-overview`
