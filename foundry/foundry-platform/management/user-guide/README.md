# Foundry Management — User Guide

This guide covers administrative tasks for Foundry Admins and Workbench Managers — Foundry onboarding, settings, Workbench provisioning, and Workbench architecture.

## Audience

| Role | Primary interest |
|------|------------------|
| Foundry Admins | Foundry onboarding, settings, organization-wide configuration |
| Workshop Admins | Workshop and Workbench provisioning |
| Workbench Managers | Workbench architecture and repository setup |

## Guide contents

| Document | Description |
