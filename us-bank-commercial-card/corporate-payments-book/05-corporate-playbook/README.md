# Part V — The Corporate's Playbook

Tells the corporate's story — how Meridian Industries takes the products Apex offers and configures them into operational programs. Starts with the seven cross-cutting administration domains, then walks through the complete operational journey for each Spend Archetype: program creation, budget assignment, eligibility, enrollment, card issuance, transaction processing, and reconciliation.
